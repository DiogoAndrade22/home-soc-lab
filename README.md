# Home SOC Lab

A home security operations lab built to practise detection engineering
and investigation, using Sysmon for endpoint telemetry and Wazuh as the SIEM.

Each writeup documents an emulated MITRE ATT&CK technique, the detection
it triggered, the supporting evidence, and the analysis.
